APW является учебным приложением, разрабатываемым в рамка курса Software Architect компании OTUS® и предназначено для автоматизации складского предприятия. Предполагается, что на основе данного проекта будет реализовано полноценное приложение по управлению складским хозяйством группы компаний Legion®. Данное приложение реализовано с использованием программных средств Microsoft® SQL Server™, Angular®, node.js® и Express®.
